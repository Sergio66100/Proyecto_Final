# 🍔 Restaurante App

## 📄 Descripción

Esta aplicación permite gestionar pedidos de un servicio de restaurante de forma sencilla.
Incluye un sistema de inicio de sesión y un CRUD completo de pedidos, donde se pueden crear, visualizar, editar y eliminar pedidos en tiempo real.

---

## 🛠️ Tecnologías utilizadas

* Frontend: React (Vite)
* Backend: Node.js + Express
* Lenguaje: JavaScript
* Comunicación: Fetch API (HTTP)
* Base de datos: En memoria (de momento)

---

## ⚙️ Requisitos previos

Para ejecutar este proyecto necesitas tener instalado:

* Node.js (versión 18 o superior)
* npm (incluido con Node.js)
* Navegador web (Chrome, Edge, etc.)

---

## 📦 Instalación

### 1. Clonar el repositorio

```bash
 https://github.com/Sergio66100/Proyecto_Final.git
```

### 2. Entrar en el proyecto

```bash
cd delivery-app
```

### 3. Instalar dependencias del backend

```bash
cd backend
npm install
```

### 4. Instalar dependencias del frontend

```bash
cd ../frontend
npm install
```

---

## ▶️ Ejecución

### 1. Iniciar backend

```bash
cd backend
node server.js
```

### 2. Iniciar frontend

```bash
cd frontend
npm run dev
```

### 3. Abrir en navegador

```
http://localhost:5173
```
